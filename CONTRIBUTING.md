# Contributing

Thank you for your interest in contributing to the XTrace SDK.

## Development Setup

```bash
git clone https://github.com/XTraceAI/xtrace-vec-sdk.git
cd xtrace-vec-sdk
pip install -e ".[dev]"
```

## Running Tests

```bash
pytest tests/
```

The crypto tests (`test_paillier_*.py`, `test_goldwasser_micali_client.py`) run fully offline and do not require an XTrace account.

## Architecture Notes

### Key Providers

`ExecutionContext` relies on a **key provider** (`KeyProvider` protocol in `src/xtrace_sdk/x_vec/crypto/key_provider.py`) to supply the AES key that protects the homomorphic secret key at rest. The SDK ships two implementations:

- **`PassphraseKeyProvider`** — derives a 256-bit key from a passphrase via scrypt. Used for local/dev workflows.
- **`AWSKMSKeyProvider`** — envelope encryption via AWS KMS. The data encryption key (DEK) is generated by KMS and only the wrapped ciphertext (EDEK) is persisted.

When creating an `ExecutionContext`, always pass an explicit `key_provider` to `ExecutionContext.create()`. The load paths (`load_from_disk`, `load_from_remote`) still accept a `passphrase` for convenience — the stored context carries the salt needed to reconstruct the passphrase-based provider. For KMS-based contexts, callers must reconstruct the `AWSKMSKeyProvider` from the stored EDEK before loading.

If you add a new key provider, implement the `KeyProvider` protocol and add tests covering create, serialize, and load round-trips.

## Code Style

This project uses [ruff](https://docs.astral.sh/ruff/) for linting and [mypy](https://mypy-lang.org/) for static type checking:

```bash
ruff check src/xtrace_sdk/
python -m mypy src/xtrace_sdk/
```

Both must pass with no errors before submitting a PR.

## Pull Requests

- Keep changes focused — one concern per PR
- Add or update tests for any changed behaviour
- Do not commit `.env` files, execution context files, or any credentials
- Update `CHANGELOG.md` under an `[Unreleased]` heading

## Reporting Issues

Open an issue on GitHub with a minimal reproducible example.
